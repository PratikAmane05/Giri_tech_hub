# PPW_2 — Day 2 Java Exercises

This document summarizes Day 2 exercises added to the repository and how to run them.

Contents (Ques_1 to Ques_19):

1. Reverse a three-digit number without loops (input: 123 -> output: 321)
2. Find first and last digit of a three-digit number
3. Sum of first and last digit of a number (input: 123 -> output: 4)
4. Find the third angle of a triangle
5. Check whether a three-digit number is a palindrome
6. Check whether a character is alphabetic
7. Input cost price and selling price; check profit or loss and amount
8. Check whether an alphabet is vowel or consonant
9. Find maximum between three numbers
10. Find minimum between three numbers
11. Print a grade using ternary operators (A/B/C/F)
12. Find total number of currency notes for an amount (500,200,100,50,20,10,5)
13. Check employee bonus eligibility (service > 5 → 5% bonus)
14. Calculate commission based on sales amount (2/5/10%)
15. Simple calculator using `switch` ( + - * / % )
16. Grade remark using `switch` (A-F)
17. Day type (weekday/weekend) using `switch` (1-7)
18. Vowel or consonant using `switch` on character
19. Convert number (1-5) to word using `switch`

Files are located in `Day_2/` inside the repository root:

- Day_2/Ques_1.java … Day_2/Ques_19.java

How to compile and run (example):

```powershell
cd PPW_1/Day_2
javac Ques_1.java
java Ques_1
```

Notes:
- Each file is a standalone console program that reads from standard input and prints results.
- If you prefer package organization, add `package` declarations and compile/run from the repo root using fully-qualified class names.
