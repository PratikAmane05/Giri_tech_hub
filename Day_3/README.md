# Day 3 — Basic Java Exercises

This folder contains Day 3 Java solutions (Ques_1 to Ques_25) and pattern programs (patt_1 to patt_10).

How to compile and run an example:

```powershell
cd PPW_1/Day_3
javac Ques_1.java
java Ques_1
```

Files included:

- Ques_1.java … Ques_25.java — core algorithm exercises (loops, arithmetic, digit operations, checks)
- patt_1.java … patt_10.java — simple pattern-printing examples (stars, numbers, characters)

Each file is a standalone console program. They accept input via `Scanner` and print results to stdout.
