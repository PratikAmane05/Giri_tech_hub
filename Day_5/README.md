# Day 5 - Array Programs

This folder contains 12 Java programs for array operations.

## Programs

1. `Ques_1.java` - Find Maximum Element in Array
   - Class: `ArrayMax`
   - Functions: `void inputArray(int arr[])`, `int findMax()`, `int findMin()`

2. `Ques_2.java` - Calculate Sum of Array Elements
   - Class: `ArraySum`
   - Functions: `void inputArray(int arr[])`, `int calculateSum()`

3. `Ques_3.java` - Calculate Average of Array Elements
   - Class: `ArrayAverage`
   - Functions: `void inputArray(int arr[])`, `double findAverage()`

4. `Ques_4.java` - Search an Element in Array
   - Class: `ArraySearch`
   - Functions: `void inputArray(int arr[])`, `boolean searchElement(int key)`

5. `Ques_5.java` - Count Even and Odd Numbers
   - Class: `ArrayEvenOdd`
   - Functions: `void inputArray(int arr[])`, `int countEven()`, `int countOdd()`

6. `Ques_6.java` - Reverse the Array
   - Class: `ArrayReverse`
   - Functions: `void inputArray(int arr[])`, `void reverseArray()`

7. `Ques_7.java` - Sort Array in Ascending Order
   - Class: `ArraySort`
   - Functions: `void inputArray(int arr[])`, `void sortArray()`

8. `Ques_8.java` - Copy Array Elements into Another Array
   - Class: `ArrayCopy`
   - Functions: `void inputArray(int arr[])`, `int[] copyArray()`

9. `Ques_9.java` - Merge Two Arrays
   - Class: `ArrayMerge`
   - Functions: `void inputArray(int arr1[], int arr2[])`, `int[] mergeArrays()`

10. `Ques_10.java` - Count Frequency of Each Element
    - Class: `ArrayFrequency`
    - Functions: `void inputArray(int arr[])`, `void countFrequency()`

11. `Ques_11.java` - Find Second Largest Element
    - Class: `ArraySecondLargest`
    - Functions: `void inputArray(int arr[])`, `int findSecondLargest()`

12. `Ques_12.java` - Check Array is Palindrome or Not
    - Class: `ArrayPalindrome`
    - Functions: `void inputArray(int arr[])`, `boolean isPalindrome()`

## Compile and Run

```powershell
cd Day_5
javac *.java
java Day_5.ArrayMax
```

Replace `ArrayMax` with any other class name to run a different program.
