# Day 2 — Basic Java Exercises

This folder contains Java solutions for Day 2 exercises (Ques_1 to Ques_19).

Problems included:

- Ques_1: Reverse a three-digit number without loops
- Ques_2: Find first and last digit of a three-digit number
- Ques_3: Sum of first and last digit of a number
- Ques_4: Find third angle of a triangle
- Ques_5: Check palindrome for a three-digit number
- Ques_6: Check whether character is alphabetic
- Ques_7: Profit or loss from cost and selling price
- Ques_8: Check vowel or consonant
- Ques_9: Maximum of three numbers
- Ques_10: Minimum of three numbers
- Ques_11: Print grade using ternary operators
- Ques_12: Count currency notes for a given amount
- Ques_13: Employee bonus eligibility
- Ques_14: Commission based on sales amount
- Ques_15: Simple calculator using switch-case
- Ques_16: Grade remark using switch-case
- Ques_17: Day type (weekday/weekend) using switch-case
- Ques_18: Vowel or consonant using switch-case
- Ques_19: Number to word (1-5) using switch-case

How these were added:

1. Collected the solution files prepared in the workspace.
2. Placed each Java file in this folder (`PPW_1/Day_2`).
3. Each file is a self-contained console program; compile and run with `javac`/`java`.

How to compile and run a file (example for `Ques_1`):

```powershell
cd PPW_1/Day_2
javac Ques_1.java
java Ques_1
```

If you prefer to keep packages, compile and run from the parent directory and use the fully-qualified class name.
